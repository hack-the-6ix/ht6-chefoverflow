Chef Overflow

This is an automation challenge. You run a five-chef kitchen where orders pour in at the reception stands, each demanding specific ingredients and preparation states. Click a chef, click a station, route them across the floor: bins, cutting boards, stoves, plating, then deliver to the matching stand.

Three expired orders ends the run. After the first minute the pace breaks any human, so the in-game API lets you spin up a kitchen agent to run the kitchen for you. The real game is the autopilot you build, not the clicks.

Top scores get auto admission to Hack the 6ix. Submit your run when the timer hits zero.

Controls: 1–5 select chefs · Space pause · Esc deselect · B boost
